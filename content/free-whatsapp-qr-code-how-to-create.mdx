# Free WhatsApp QR Code: How to Create One (No App Required)

A WhatsApp QR code opens a chat with your number—optionally with a pre-filled message like "Hi, I'd like to order" or "Get support." You can create one in under a minute, free, with no app or account sign-up. Great for support, orders, events, and contactless contact.

## What You Need

Your phone number with country code (e.g. +1, +44, +39). Optional: a default message that opens in the chat when someone scans (e.g. "Hi, I'd like to place an order" or "I need help with my order"). The message is optional but useful so customers know what to send.

Enter your full number with country code (e.g. +1 555 123 4567). The country code is required so the link works for everyone. Add an optional default message—e.g. "Order here" or "Support request"—so scanners see a suggested first line. Click generate and the QR code appears.

For digital use (email, social, screens), 256px is usually enough. For print—menus, table tents, flyers—use at least 512px so it scans reliably at a distance. Download PNG or SVG and add it to your design.

## Do Guests Need an App?

Guests need WhatsApp installed (or they can open the link in a browser where WhatsApp Web is available). When they scan, their device will open WhatsApp with your number and the pre-filled message. If they don't have WhatsApp, they'll be prompted to install it or open in browser. You don't need WhatsApp Business to use a WhatsApp QR code; a regular WhatsApp number is enough.

## FAQ

**Do I need WhatsApp Business to use a WhatsApp QR code?**  
No. A standard WhatsApp number is enough. You enter your number (with country code) in the generator; the code opens a chat with that number. WhatsApp Business is optional and useful if you want business profile features, but not required for the QR code.

**Can I add a default message to the QR code?**  
Yes. Most WhatsApp QR generators let you add an optional pre-filled message. When someone scans, WhatsApp opens with your number and that message ready to send. Useful for "Order here," "Support," or "I have a question."

**What size should the WhatsApp QR code be for print?**  
Use at least 512px for print (menus, table tents, flyers) so it scans reliably at arm's length or farther. For digital use, 256px is usually enough.

**Do guests need an app to scan it?**  
They need WhatsApp installed (or they can open the link in a browser). The QR code opens a wa.me link that launches WhatsApp with your number. If they don't have the app, they'll be prompted to install it or continue in the browser.
